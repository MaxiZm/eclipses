# Eclipse Simulator

Простой одностраничный сайт для визуальной симуляции солнечных затмений с управляемыми параметрами орбитальной геометрии.

## Что умеет

- Изменяемые параметры:
  - эклиптическая долгота восходящего/нисходящего узлов;
  - эклиптическая долгота Солнца;
  - положение Луны относительно узла;
  - расстояние до Луны (перигей ↔ апогей);
  - положение наблюдателя на Земле.
- 6 режимов отображения:
  1. Меркатор;
  2. Глобулярный вид;
  3. Вид от полюса эклиптики;
  4. Вид от наземного наблюдателя;
  5. Вид от Солнца;
  6. Поперечный разрез системы Солнце–Луна–Земля.

## Запуск

Откройте `index.html` в браузере или поднимите локальный статический сервер:

```bash
python3 -m http.server 4173
```

После этого перейдите на `http://localhost:4173`.
